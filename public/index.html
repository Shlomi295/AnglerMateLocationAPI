Angler Mate Location API

Getting started
The API requires an API key in the headers: 
Key: authorization  Value: {apiKey}

GET ALL Records: Call => https://amlocatapi.us-south.cf.appdomain.cloud/location/


GET BY ID: Call => https://amlocatapi.us-south.cf.appdomain.cloud/location/{ID} 

Example: https://amlocatapi.us-south.cf.appdomain.cloud/location/60991ca5f025547818598ab0

GET BY NAME: Call => https://amlocatapi.us-south.cf.appdomain.cloud/location/name/{NAME} 

 Example: https://amlocatapi.us-south.cf.appdomain.cloud/location/name/lungfish

POST INSERT A RECORD: Call => https://amlocatapi.us-south.cf.appdomain.cloud/location/name/{NAME} 

 Example: 
                   POST TO -> https://amlocatapi.us-south.cf.appdomain.cloud/location/

                   This Json structure:

                    {
                    "fish": "Karp",
                    "lat": 58.15,
                    "long": 758.11,
                    "url": "https://scienceinpoland.pap.pl/sites/default/files/styles/strona.png"
                    }
